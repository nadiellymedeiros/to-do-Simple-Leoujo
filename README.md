# Descrição

Esse projeto foi feito como treinamento pessoal. Consiste em uma aplicação web de uma lista de tarefas. Pode-se editar e apagar a tarefa desejada.

# Features

- [x] Adicionar tarefa
- [x] Listar tarefas
- [x] Editar e apagar tarefa
